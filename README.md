# 国民的RPGキャラクターでひらがなを学習するゲーム
React + TypeScript + Vite  
`npm run build` でビルドしたファイルで遊んでください。  

実際に動いているものを手っ取り早く見たい場合  
https://endoyuki.jp/private/poxxmon/  
(endo/yuki1005)

クローンしただけでは、画像ファイルが不足しているため以下をダウンロード。  

- http://endoyuki.jp/download/zukan.zip 
- http://endoyuki.jp/download/common.zip

それぞれZipのパスワードは「monster」です。  

解凍した「zukan」と「common」を`src/asests/images/`に配置し、ビルドすれば動作します。
